# Electric Vehicle Data Analysis

This project analyzes Electric Vehicle population data to understand
adoption trends, electric range distribution, and predictive modeling.

## Files Included
- Electric_Vehicle_Data_Analysis.pdf
- Jupyter Notebook (.ipynb)

## Analysis Covered
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Linear Regression Model

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Submission Note
The PDF report contains the full analysis and visualizations.
The Python code is provided for reference and reproducibility.
